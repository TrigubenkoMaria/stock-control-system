# Warehouse Management System (WMS) 

Веб-система для автоматизации складского учета, контроля остатков и управления инвентарем. Разработана в качестве дипломного проекта.

## Основные возможности
*   **Учет на нескольких складах**: Отслеживание остатков товаров в разрезе конкретных мест хранения.
*   **Бизнес-логика**: Автоматический пересчет остатков при совершении транзакций (Приход/Расход) с защитой от отрицательных значений.
*   **Маркировка QR-кодами**: Автоматическая генерация QR-кода для каждого товара со ссылкой на его карточку для быстрой инвентаризации.
*   **Telegram-уведомления**:В системе реализован модуль мгновенных уведомлений о дефиците товаров. По умолчанию данный функционал отключен в коде (`models.py`) для обеспечения полной автономности системы и исключения задержек при отсутствии интернет-соединения с серверами.
*   **Работа с данными**: Импорт и экспорт через Excel (.xlsx).
*   **Dashbord**: Наглядная статистика дефицита товаров и последних операций на главной странице.
*   **Безопасность**: Скрытая панель администратора и разграничение доступа.

## Установка и запуск
1. Настройте виртуальное окружение.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте .env в корне проекта и заполните переменными окружения.
4. Уведомления от бота: в методе `save` модели `Transaction` **раскомментируйте** блок кода, отвечающий за вызов функции `send_deficit_notification`. 
5. Примените миграции: `python manage.py migrate`.
6. Создайте админа: `python manage.py createsuperuser`.
7. Запустите сервер: `python manage.py runserver`.
